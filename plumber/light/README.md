# Plumber Light Template

A clean, professional single-page website template for plumbing businesses.
Drop-in ready. Clone for every plumber.

## How to Use (2 minutes per client)

1. Copy the folder: `cp -r plumber-light-template "plumber-<business-name>"`
2. Replace `<!-- BUSINESS NAME -->` and `<!-- TAGLINE -->` in index.html
3. Replace the phone number (search `(518) 555-1234`)
4. Drop in their hero image as `assets/hero.jpg`
5. Add real photos to the gallery section
6. Deploy to GitHub Pages

## What's Included

- 5 sections: Hero, Services, About, Gallery, Contact
- Config block at top (CSS variables for brand colors)
- Mobile-responsive (burger menu on phones)
- Contact form (needs backend or form service)
- SEO-ready meta tags
- Google Stars rating display
- 1 custom Nano Banana Pro hero image
